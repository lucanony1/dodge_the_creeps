# Dodge the creeps
what is the game about and how to play it:
- Dodging creeps who come from random positions and random angles
- Use the arrows to move around and avoid the monsters
- Click on the ▶️ to start the game.
The highscore is only yours, not worldwide.

NOTICE: If you restart a game, the highscore will also refresh itself


TIP: the creeps do not follow you, so you will not have billions after you
